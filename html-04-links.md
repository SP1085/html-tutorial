# ลิงก์และ Anchor

## 🔹 ลิงก์ (Hyperlink) คืออะไร?

**ลิงก์** หรือ **Hyperlink** คือองค์ประกอบในหน้าเว็บที่สามารถคลิกได้ เพื่อไปยัง:
- หน้าอื่นภายในเว็บไซต์เดียวกัน
- เว็บไซต์ภายนอก
- จุดใดจุดหนึ่งในหน้าเดียวกัน
- ไฟล์ดาวน์โหลด เช่น PDF, รูปภาพ

HTML ใช้แท็ก `<a>` (**anchor**) สำหรับสร้างลิงก์

---

## 🔸 โครงสร้างแท็ก `<a>`

```html
<a href="URL">ข้อความที่คลิกได้</a>
```

- `href` (Hypertext Reference) คือ URL หรือที่อยู่ปลายทาง
- ข้อความระหว่าง `<a>...</a>` คือสิ่งที่ผู้ใช้จะคลิก

---

## 🔸 ตัวอย่างลิงก์

### 🌐 ลิงก์ไปยังเว็บไซต์อื่น:

```html
<a href="https://www.google.com">ไปที่ Google</a>
```

### 🏠 ลิงก์ภายในเว็บไซต์:

```html
<a href="about.html">เกี่ยวกับเรา</a>
```

### 📄 ลิงก์ไปยังไฟล์:

```html
<a href="document.pdf" download>ดาวน์โหลดเอกสาร</a>
```

### 🔗 ลิงก์เปิดในแท็บใหม่:

```html
<a href="https://example.com" target="_blank">เปิดในแท็บใหม่</a>
```

- `target="_blank"` คือการสั่งให้ลิงก์เปิดในหน้าต่างใหม่

---

## 🔸 ลิงก์ภายในหน้า (Anchor Link)

ใช้เพื่อเลื่อนลงไปยังส่วนต่าง ๆ ของหน้าเดียวกัน

### ตัวอย่าง:

```html
<a href="#section2">ไปที่ส่วนที่ 2</a>

<h2 id="section2">ส่วนที่ 2</h2>
<p>เนื้อหาในส่วนที่ 2</p>
```

- ต้องมี `id` ตรงกับ `href` ที่มี `#`

---

## 🧠 สรุป

| รูปแบบลิงก์ | ตัวอย่าง | หมายเหตุ |
|-------------|----------|----------|
| ลิงก์ภายนอก | `<a href="https://...">` | ใช้ URL แบบเต็ม |
| ลิงก์ภายใน | `<a href="about.html">` | ใช้ชื่อไฟล์ภายในเว็บ |
| เปิดแท็บใหม่ | `target="_blank"` | ใช้คู่กับ href |
| ลิงก์ภายในหน้า | `<a href="#id">` | ต้องมี `<id>` ตรงกันในเป้าหมาย |
