# บทที่ 1: เว็บทำงานอย่างไรและเครื่องมือที่จำเป็น
## 1.1 บทบาทของภาษาหลัก
### บทบาทและความแตกต่างของภาษาหลัก 3 ภาษา

บทบาทของ HTML, CSS, และ JavaScript เป็นสิ่งสำคัญที่สุดในการเริ่มต้นพัฒนาเว็บไซต์ ภาษาทั้งสามนี้ทำงานร่วมกัน แต่มีหน้าที่ที่แตกต่างกัน

| ภาษา | ชื่อเต็ม | บทบาทหลัก |
| :-------------------- | :------------------ | :------------------ |
| HTML | HyperText Markup Language | โครงสร้างและเนื้อหา กำหนดองค์ประกอบต่างๆ ของหน้าเว็บ (เช่น หัวข้อ, ย่อหน้า, รูปภาพ, ตาราง) เป็นเหมือนโครงร่างพื้นฐานของข้อมูล |
|CSS |	Cascading Style Sheets | การจัดรูปแบบและรูปลักษณ์ กำหนดวิธีการแสดงผลของเนื้อหา HTML (เช่น สี, ฟอนต์, ขนาด, ตำแหน่ง, การจัดวางหน้า)
|JavaScript |	JavaScript | การโต้ตอบและการทำงาน เพิ่มฟังก์ชันการทำงานแบบไดนามิกและการโต้ตอบกับผู้ใช้ (เช่น ปุ่มที่คลิกได้, ภาพเคลื่อนไหว, การตรวจสอบข้อมูลในฟอร์ม)
